# 💰 Controle de Despesas do joao

Projeto simples para controle de gastos pessoais.

## Funcionalidades
- Adicionar despesas
- Ver saldo total
- Ver receitas e despesas
- Remover e editar itens
- Destaque para valores altos

## Tecnologias
- HTML
- CSS
- JavaScript

## Como rodar
Abra o index.html com Live Server
